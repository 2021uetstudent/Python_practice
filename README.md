## 📂 Project List & Descriptions

### 🔹 Variables & Logic Practice

Here are the programs I practiced to build strong fundamentals with variables, data types, and control flow:

#### 1. Student Enrollment Record (Data Analyst Course)
- **Description:** A program to manage and print student enrollment details. It practices taking user inputs, validating strings, and organizing different data types (Strings, Integers, Booleans) into a structured format.
- **Key Concepts Used:** Variables initialization, User Inputs, Type Casting, `for` loops, Conditional statements (`if/else`, `break`).

#### 2. Circle Stats Calculator
- **Description:** A math-based program that takes the radius of a circle as input and automatically calculates its Area and Circumference.
- **Key Concepts Used:** Constants, Arithmetic Operators, Math formulas integration.

#### 3. Find the Middle Number
- **Description:** A logical program that takes three different numbers as input and uses comparison logic to filter out the exact middle number.
- **Key Concepts Used:** Multiple variable comparisons, Logical operators (`and`, `or`), Nested conditional logic.

#### 4. Digit Extract Game
- **Description:** A clever math-based puzzle/game that takes a multi-digit number (e.g., a 3-digit number like 123) and breaks it down to extract each individual digit into its own variable.
- **Key Concepts Used:** Modulo operator (`%`), Floor division (`//`), Variable re-assignment.

### 🔹 String Manipulation Practice

Here are the programs I practiced to strengthen my understanding of Python strings — a core skill for cleaning and processing real-world data:

#### 1. Word & Character Frequency Counter
- **Description:** Counts how many times each unique word and each unique character appears in a given paragraph.
- **Key Concepts Used:** `split()`, dictionaries, `collections.Counter`, text cleaning (lowercasing, punctuation removal).

#### 2. Vowel & Consonant Counter
- **Description:** Counts the total number of vowels and consonants present in a string.
- **Key Concepts Used:** Looping through strings, conditional checks, `in` operator.

#### 3. Replace All Vowels with `#`
- **Description:** Scans a string and replaces every vowel with the `#` character.
- **Key Concepts Used:** String iteration, conditional replacement, string concatenation.

#### 4. String Reverse
- **Description:** Reverses a given string using multiple approaches (loop-based and slicing).
- **Key Concepts Used:** Negative indexing, slicing (`[::-1]`), loops.

#### 5. Reverse the Word Order
- **Description:** Takes a sentence and reverses the order of the words (not the characters), e.g., `"I love Python"` → `"Python love I"`.
- **Key Concepts Used:** `split()`, list reversal, `join()`.

#### 6. Palindrome Checker
- **Description:** Checks whether a given word or phrase reads the same forwards and backwards.
- **Key Concepts Used:** String reversal, comparison operators.

#### 7. Sort All Characters
- **Description:** Sorts all characters of a string alphabetically.
- **Key Concepts Used:** `sorted()`, `join()`, string-to-list conversion.

#### 8. Anagram Checker
- **Description:** Checks whether two given words are anagrams of each other (contain the exact same letters).
- **Key Concepts Used:** Sorting, `collections.Counter`, string comparison.

#### 9. Remove Duplicate Characters from a String
- **Description:** Removes repeated characters from a string while preserving the first occurrence and original order.
- **Key Concepts Used:** Sets, dictionaries (`dict.fromkeys()`), iteration.

#### 10. Title Case / Sentence Case Converter & Uppercase Converter
- **Description:** Converts a given string into Title Case, Sentence Case, or fully Uppercase format.
- **Key Concepts Used:** `title()`, `capitalize()`, `upper()` string methods.

#### 11. Maximum Character (Lexicographically) from the String
- **Description:** Finds the character with the highest lexicographical (alphabetical) value in a string.
- **Key Concepts Used:** `max()` function, character comparison, ASCII ordering.

#### 12. Count Total Number of Characters in a String (Without `len()`)
- **Description:** Calculates the length of a string manually, without using Python's built-in `len()` function — solved using a loop, recursion, and the `count("")` string-method trick.
- **Key Concepts Used:** Loops, recursion, `str.count()` method.
